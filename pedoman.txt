=== DATA FLOW ===

1. DRAG DRAG: palette → startDrag() → ACTION_DRAG_STARTED
   → ENTERED/LOCATION → shadow ditambahkan
   → ACTION_DROP → removeWidget(shadow), createView(), addWidget()
   → ACTION_DRAG_ENDED → removeWidget(shadow) safety

2. RESIZE: touch handle → onInterceptTouchEvent hitung jarak 8 handle
   → ACTION_MOVE → resizeSelectedView(dx, dy)
   → ACTION_UP → commit width/height ke layoutParams + attributeMap (dp)
   → Gate: isResizeEnabled (default false)

3. NUDGE: onKeyDown → nudgeSelectedView(dx, dy)
   → suppressLayout → sv.layout() → update margins + attributeMap

4. EDIT XML: XmlLayoutGenerator.generate() → EditXMLActivity
   → return XML → loadLayoutFromParser() rebuild tree

5. UNDO/REDO: UndoRedoManager simpan XML string
   → loadLayoutFromParser(xmlString) restore

6. SAVE: XmlLayoutGenerator().generate(editor, true) → file .xml

=== KNOWN BUG FIXES ===

1. ConstraintLayout crash (LinearLayout.LayoutParams cast)
   → addWidget() detect parent LayoutParams type, create via reflection

2. Shadow persist setelah drag
   → removeWidget(shadow) di ACTION_DRAG_ENDED + resize start

3. NPE showDefinedAttributes
   → viewAttributeMap[target] ?: return

4. java.lang.Object di XML output
   → XmlLayoutGenerator peek() skip class == "java.lang.Object"

5. Shadow layout params corrupted
   → if (view !== shadow) skip layout params change
